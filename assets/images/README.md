# Image Placeholders

Esta pasta está reservada para imagens e demonstrações do protótipo físico.

Sugestões de arquivos:

| Arquivo | Conteúdo sugerido |
| --- | --- |
| `robot-front.jpg` | Foto frontal do robô, mostrando o sensor ultrassônico e sensores de borda |
| `robot-top.jpg` | Foto superior da montagem, mostrando ESP32, L298N e fiação |
| `wiring-diagram.png` | Diagrama de ligação elétrica |
| `arena-test.jpg` | Foto da arena ou superfície de teste |
| `demo.gif` | Demonstração curta do robô buscando ou avançando |

Ao adicionar imagens, prefira nomes curtos, boa iluminação e enquadramento que mostre o hardware com clareza.
